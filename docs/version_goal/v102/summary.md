# v102 版本更新摘要

本版本主要包含番茄钟全屏聚焦体验优化、任务备注富文本与附件支持、清单撤销重做功能集成，以及 Android 构建兼容性优化。

## 新增功能
1. **任务备注富文本编辑器**
   - 任务备注输入框全面升级为富文本编辑器，支持 Markdown 语法编辑和渲染（如标题、粗体、斜体、列表等）。
2. **任务附件管理**
   - 任务详情中新增附件插入支持，用户可以向任务添加图片、文档等各类文件，并支持附件的预览与删除。
3. **清单操作撤销与重做**
   - 清单的创建、修改和删除操作已接入全局撤销重做系统。
   - 删除清单时，该清单下的所有任务均会自动归档或备份；撤销删除清单时，系统能自动恢复清单并级联找回原来属于该清单的所有任务。
   - 撤销重做历史页面新增清单操作类型记录的渲染和过滤统计。

## 优化与改进
1. **番茄钟全屏页面体验重构**
   - 全屏专注页面全面优化，支持三种显示模式的交互切换（点击背景或特定按钮即可循环/回退）：
     - **白色主题（默认）**：纯白底色，展示完整计时环、关联任务、快捷设置和全部控制按钮。
     - **黑色主题**：纯黑底色与浅色文字，适配暗光环境，保留完整计时及控制功能。
     - **沉浸模式**：纯黑极简底色，隐藏除核心计时区域外的所有元素（如导航栏、控制按钮、快捷设置等）。
   - 沉浸模式下支持左右滑动切换两种计时界面样式：
     - **圆环进度条样式**：带有白色极简环形进度条、大字号时间和当前状态展示。
     - **纯数字样式**：无进度环，仅以极简超大字号显示剩余时间及状态。
   - 修复并优化了专注进度指示器的同步与显示逻辑。

## 问题修复
1. **Android 构建环境兼容性修复**
   - 升级 `Android Gradle Plugin (AGP)` 版本至 `8.9.1`，解决项目构建时发生的第三方 AAR 元数据依赖校验冲突问题。
   - 优化了 Android 下文件/媒体库读取权限的相关配置，保证任务备注插入附件功能正常运行。
