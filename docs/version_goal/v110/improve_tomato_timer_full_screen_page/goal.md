# 番茄钟全屏聚焦页面优化目标

## 1. 界面主题与显示模式切换
优化后的全屏番茄钟页面支持以下三种显示模式，点击背景或特定按钮可进行循环/回退切换：
- **默认白色主题 (`default(white_theme).png`)**：
  - 启动进入时的默认状态，纯白背景。
  - 显示所有功能模块：顶部导航栏（含退出按钮、常亮开关）、关联任务展示、中间环形计时器、快捷时间设置、周期小圆圈、底部主控制栏。
  - 点击页面背景切换至 **黑色主题**。
  - 点击左上角返回按钮退出全屏页面，返回至番茄钟主页。

- **黑色主题 (`black_theme.jpg`)**：
  - 纯黑背景，适配暗色/AMOLED 屏。
  - 显示与白色主题相同的所有功能模块，但采用白色/浅灰色文字与图标。
  - 点击页面背景切换至 **沉浸模式**。
  - 点击左上角返回按钮退回到 **默认白色主题**。

- **沉浸模式 (`immerse_view1.jpg` & `immerse_view2.jpg`)**：
  - 隐藏所有功能模块（无导航栏、无控制按钮等），仅保留核心的计时区域，背景为纯黑色。
  - 支持左右滑动切换两种不同的沉浸式计时样式：
    - **样式一 (圆环进度条模式)**：显示白色环形进度条、大字号时间及当前状态（如“专注中”）。
    - **样式二 (纯数字模式)**：无圆环进度条，仅以极简超大字号显示剩余时间及当前状态。
  - 点击屏幕任意背景区域切换回 **黑色主题**（以便重新显示控制按钮进行暂停/放弃等操作）。

## 2. 代码架构优化
- 计时区域（Timer Area）与功能模块（Functional Modules）解耦，提高代码复用性：
  - 提取计时区域为可替换的独立组件，根据不同的显示模式切换其样式（圆环进度条或纯数字）。
  - 功能模块（如关联任务、快捷设置、控制按钮等）在非沉浸模式下正常渲染，在沉浸模式下隐藏。
